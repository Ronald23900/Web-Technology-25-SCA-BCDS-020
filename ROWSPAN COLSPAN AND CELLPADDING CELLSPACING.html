Table spacing demo

This example shows a table with cellpadding (space inside cells) and cellspacing (space between cells). Below it is a table without those attributes for comparison.

With cellpadding="12" and cellspacing="8"

Header 1 Header 2
Cell A1 Cell A2

Without cellpadding / cellspacing (default)

Note: In HTML5 the cellpadding and cellspacing attributes are considered presentational and CSS (for example, padding on td and border-spacing) is preferred. However, these attributes still work in most browsers and are shown here for learning.
